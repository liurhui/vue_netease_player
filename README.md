# vue_netease_player
vuecli4网易云web播放器
备注
    运行成功后，播放歌曲 如果安装过 Internet Download Manager(IDM)下载器,会提示下载音乐
    解决办法：打开IDM 在菜单栏的Downloads/Options  把 关联的 浏览器 Google Chrome 取消打勾 保存。
